## Getting Started

All features can be accessed through the Osiris menu in-game. Press the designated key to open the menu and navigate through the tabs:

1. **Combat** - Weapon-related features
2. **HUD** - Timer and alert customizations
3. **Visuals** - Visual effects and player information
4. **Sound** - Audio visualization tools

## Configuration

All settings are automatically saved to your active configuration file (`default.cfg`). You can restore defaults at any time using the "Restore Defaults" button in the menu navbar.

## Quick Reference

### Combat Features
- No Scope Inaccuracy Visualization for sniper rifles

### HUD Features
- Bomb Timer with explosion countdown and site indicator
- Defusing Alert showing defuse progress
- Bomb Plant Alert indicating if plant can complete before round end
- Killfeed Preserver to keep your kills visible
- Post-round Timer display

### Visual Features
- **Player Info In World** - Show health, weapons, position, and status icons
- **Outline Glow** - Highlight players, weapons, grenades, bombs, and hostages
- **Model Glow** - Apply colored glow effects to player and item models
- **Viewmodel** - Customize your weapon viewmodel FOV

### Sound Features
- Player Footstep Visualization
- Bomb Plant Sound Visualization
- Bomb Beep Sound Visualization
- Bomb Defuse Sound Visualization
- Weapon Scope Sound Visualization
- Weapon Reload Sound Visualization
